# Landing page - Grúa Amigo

Landing page estática inspirada en sitios de remolques/auxilio vehicular 24/7.

## Ejecutar localmente

```bash
python3 -m http.server 4173
```

Abrir: `http://localhost:4173`

## Publicar en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir este contenido a la rama `main`.
3. En `Settings > Pages`, seleccionar:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` y carpeta `/ (root)`
4. Guardar y esperar el deploy.

La URL pública quedará en el formato:
`https://<usuario>.github.io/<repositorio>/`
